Question Formatting Guide

Formatting Notice: Complex structures such as tables, matching pairs, and ordering questions are supported but may not work perfectly in all document styling variations. Please review the Live Preview carefully to ensure all items are parsed correctly before downloading the QTI package.

1. Aiken Format (Multiple Choice Only)

Paste questions directly from Google Docs or PDFs. Each choice must start with a letter and parenthesis/period, followed by the answer statement.

What is the capital of Texas?
A) Austin
B) Houston
C) Dallas
ANSWER: A

2. text2qti Markdown Format (All Types)

Use simple markdown keywords to define titles, points, and varied question types.

Multiple Choice & True/False: Mark the correct answer with an asterisk (`*`):

Title: MC Example
Points: 1
1. Choose the primary cloud provider.
a) AWS
b) Azure
*c) Google Cloud
... Feedback for choice c

Multiple Answers (Checkboxes): Use `[*]` for correct and `[ ]` for incorrect choices:

Points: 2
2. Select all cloud databases.
[*] DynamoDB
[ ] EC2
[*] Cloud Spanner

Fill in the Blank: Use asterisk `*` lists to specify all acceptable answers:

3. What is the abbreviation for Texas State Technical College?
* TSTC
* T.S.T.C.

Essay Question: Add three underscores `___` on a new line below the prompt:

4. Explain the benefits of continuous integration in software delivery.
___

Numerical Range: Specify exact values and margins (`+-`), or range brackets `[min, max]`:

5. Solve for x: 5 * 5.
= 25 +- 0.1

3. Spreadsheet Grid Importer

Copy-paste headers and cells directly from Excel or Google Sheets. The grid maps headers automatically. Multiple Answer correct options can be comma-separated list like `A, C`.
