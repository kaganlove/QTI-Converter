# QTI Converter for Canvas LMS

A modern, high-fidelity static web application designed specifically for **Instructional Designers at TSTC** to easily convert exam questions from Google Docs, Excel/Google Sheets, or PDFs into Canvas-compatible QTI 1.2 ZIP packages. 

This tool runs **entirely client-side** in the browser, meaning your question banks are processed locally and securely with no data sent to any backend servers.

---

## Key Features

1. **Text Importer**:
   - Supports copy-pasting standard **Aiken format** (Multiple Choice test banks).
   - Supports **text2qti Markdown format** which enables Multiple Choice, True/False, Multiple Answer (checkboxes), Short Answer (Fill-in-the-blank), Essays, File Uploads, and Numerical questions.
2. **Spreadsheet Grid Importer**:
   - Easily copy-paste cell blocks from Google Sheets or Microsoft Excel directly into the interactive grid.
   - Automatically parses columns (Question Text, Type, Points, Correct Answers, Choices, Feedback).
3. **Live Preview & Validation**:
   - Compiles and renders questions in real-time.
   - highlights parsing errors (e.g., missing correct answers, invalid points) before you export.
4. **Inline Editor**:
   - Directly tweak question prompts, points, or choices by clicking on them inside the live preview.
5. **Options Panel**:
   - Option to customize the **Question Bank Name** (Quiz Title).
   - Configure Canvas-specific settings: Shuffle Answers, Show Correct Answers, One Question at a Time, and Locked Navigation.
6. **TSTC Royal Blue Theme**:
   - Customized with TSTC's primary Royal Blue brand colors for a clean, cohesive user experience.

## How to Import into Canvas

Once you download your QTI ZIP package, follow these steps to upload it into your Canvas course as a question bank:

1. Log in to **Canvas** and open your course.
2. Navigate to **Settings** in the course navigation menu.
3. Select **Import Course Content** from the right-hand sidebar.
4. In the **Content Type** dropdown, select **QTI .zip file**.
5. Click **Choose File** and select the ZIP package generated by this converter.
6. (Optional) Check "Import assessment as New Quizzes" if you prefer New Quizzes, or leave unchecked for classic Question Banks.
7. Click **Import**. The question bank will appear in your **Quizzes > Manage Question Banks** page.

## Question Formatting Rules

### 1. Aiken Format (Multiple Choice Only)
Ideal for basic multiple-choice question lists from Word/Google Docs. Each question starts with a prompt on a single line, choices begin with a letter followed by a parenthesis or period, and the correct answer is declared with the `ANSWER:` prefix.

```text
What is the primary branding color of TSTC?
A) Royal Blue
B) Gold
C) Green
D) Silver
ANSWER: A
```

### 2. text2qti Markdown Format (All Types)
A powerful markdown syntax that supports multiple question types, points, and feedbacks.

#### Multiple Choice & True/False
Prefix the correct choice letter with an asterisk `*`.
```text
Title: Brand Identity
Points: 1
1. What is the main color in the TSTC logo?
*a) Royal Blue
b) Black
c) White
... Royal Blue (#1A73E8) is the primary color.
```

#### Multiple Answer (Select all that apply)
Use `[*]` for correct selections and `[ ]` for incorrect ones.
```text
Points: 2
2. Which files are inside a Canvas QTI package? (Select all that apply)
[*] imsmanifest.xml
[ ] config.json
[*] assessment_meta.xml
```

#### Short Answer / Fill in the Blank
List all acceptable answers on lines starting with an asterisk `*`.
```text
3. What does LMS stand for?
* Learning Management System
* Learning Management Systems
```

#### Essay Response
Place three or more underscores `___` on a line below the prompt.
```text
4. Describe your process for mapping course competencies.
___
```

#### File Upload
Place three or more carets `^^^^` on a line below the prompt.
```text
5. Upload a PDF of your syllabus map.
^^^^
```

#### Numerical Ranges
Define exact answers with margin tolerances (`+-`) or bracket ranges `[min, max]`.
```text
6. What is the result of 100 divided by 4?
= 25 +- 0.1
```

### 3. Spreadsheet Layout
For Excel or Google Sheets, organize columns with the following headers:
- `Question Text`
- `Type` (MC, TF, MA, FITB, Essay, Num)
- `Points`
- `Correct Answer(s)` (for MA, separate correct letters with commas, e.g., `A, C`)
- `Choice A`
- `Choice B`
- `Choice C`
- `Choice D`
- `Explanation Feedback`

## Hosting on GitHub Pages

Since this is a client-side single-page application with zero server dependencies, hosting it on GitHub Pages is free and takes just seconds:

1. Create a new repository on your GitHub account (`kaganlove`) named **QTI Converter**.
2. Commit and push the files from this directory to the `main` branch.
3. On GitHub, go to your repository **Settings > Pages**.
4. Under **Build and deployment**, select **Deploy from a branch** and choose **main** (root folder).
5. Click **Save**. Your converter will be live at `https://kaganlove.github.io/QTI-Converter/` in a few minutes!
